1.撤销commit：git reset --soft HEAD^
	--mixed 不删除工作空间改动代码，撤销commit，并且撤销git add . 操作
	--soft  不删除工作空间改动代码，撤销commit，不撤销git add . 
	--hard	删除工作空间改动代码，撤销commit，撤销git add .
